# ARKA — Product & Execution Document

## 1. Product Vision

ARKA is not an app to “count days without smoking”.

It is an app designed to **restore agency over cannabis consumption**.

Its purpose is to help users:

- understand what triggers their consumption
- detect when they are entering a risk zone
- introduce friction before consuming
- tolerate urges without acting
- rebalance their reward system
- see real progress across multiple dimensions

---

---

### Result

ARKA should feel like:

> Kwit + behavioral coach + relapse radar + personal dopamine lab

## 3. Core Systems

ARKA is built on 6 main systems.

### 3.1 Progress System

Converts recovery into visible progress.

Categories:

- mental clarity
- sleep
- emotional regulation
- motivation and energy
- memory and focus
- physical health
- finances
- relationships
- self-control
- dopamine balance
- resilience

Each category includes:

- time-based milestones
- behavior-based milestones
- dynamic scoring

---

### 3.2 Risk System

Calculates a real-time **Risk Score** based on:

- craving
- anxiety / stress
- sleep
- context
- time of day
- history

---

### 3.3 DOPAMINE System

Behavioral framework:

- Data
- Objectives
- Problems
- Abstinence
- Mindfulness
- Insight
- Next Steps
- Experiment

---

### 3.4 Self-Binding System

Introduces friction before consumption.

Types:

- Time (delays, schedules)
- Space (location, availability)
- Meaning (contracts, intent, accountability)

---

### 3.5 Intervention System (JITAI)

Real-time actions when risk is high.

Includes:

- alerts
- suggested actions
- re-evaluation

---

### 3.6 Recovery System

Prevents collapse after relapse.

Includes:

- non-judgmental logging
- analysis
- recovery plan
- control restoration

---

### 3.7 Pleasure–Pain Balance System

Core feature.

Before consumption:

- expected pleasure
- expected relief
- future cost

After:

- actual pleasure
- duration
- real cost

## 6. MVP Definition (Locked Scope)

### Included Features

- basic onboarding
- check-in
- consumption tracking
- simple risk score
- interventions (3 types)
- progress (3 categories)
- basic pleasure–pain balance
- simple recovery mode

---

### Excluded

- full DOPAMINE system
- advanced self-binding
- social features
- advanced insights
- personalization
- AI coach

---

### MVP Goal

Allow the user to resist consumption at least once per day.
